# 📊 AI Data Analyst V2

An AI-powered data analysis application built with **Python**, **Streamlit**, and **Plotly** that automatically analyzes any CSV, Excel, or JSON dataset.

The application provides dataset profiling, descriptive analytics, interactive dashboards, AI-powered querying, forecasting, anomaly detection, driver analysis, root cause analysis, and executive reporting through a modern modular interface.

# 📥 Installation

Clone the repository

```bash
git clone https://github.com/avviiiral/AI_Data_Analyst_V2.git
```

Move into the project

```bash
cd AI_Data_Analyst_V2
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
